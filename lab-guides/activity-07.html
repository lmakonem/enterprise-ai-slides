üìù Step-by-Step Instructions

Part 1: Understand RAG (10 min)

Traditional LLMs (like ChatGPT) only know what they were trained on (public internet data up to a certain date). RAG fixes this by:
1. Retrieving: Finding relevant snippets from your private documents.
2. Augmenting: Adding those snippets to the prompt.
3. Generating: Asking the LLM to answer using only those snippets.

Part 2: Setup Environment (15 min)

  1. Create a Folder: Name it rag-demo.
  2. Copy Files: Copy app.py, requirements.txt, and company_policy.txt from the lab materials into this folder.
  3. Install Libraries:
    Open your terminal/command prompt in this folder and run:
    bash pip install -r requirements.txt
    (Note: This might take a few minutes as it installs langchain, chromadb, and torch).

Part 3: Configure Token (5 min)

  1. Get Token: Use the same hf_... token from Activity 6.
  2. Set Environment Variable:
    • Windows (PowerShell): $env:HF_TOKEN="your_token_here"
    • Mac/Linux: export HF_TOKEN="your_token_here"
    • Or (Not Recommended for Prod): Hardcode it in app.py temporarily if you struggle with env vars.

Part 4: Run the App (20 min)

  1. Run the Script:
    bash python app.py
  2. Wait for Launch: You will see a local URL (e.g., http://127.0.0.1:7860). Open it in your browser.

Part 5: Test the Bot (20 min)

Try these questions and observe the answers:
1. Basic Retrieval: "How many vacation days do I get?"
(Expected: 15 days)
2. Specific Rule: "Can I wear jeans to a client meeting?"
(Expected: No, client meetings require Business Professional)
3. Negative Test: "What is the capital of France?"
(Expected: The bot might try to answer based on the policy, or fail gracefully. Since the context doesn't contain geography, a good RAG system should say "I don't know" or hallucinate less than a raw model.)

Part 6: Modify the Data (20 min)

  1. Edit company_policy.txt: Change the remote work policy to "Fridays are remote-only."
  2. Restart the App: Stop the script (Ctrl+C) and run python app.py again.
  3. Ask Again: "Can I work from home on Friday?"
    (Expected: Yes, based on your new data).
