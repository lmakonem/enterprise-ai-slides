üìß Communication (8 Prompts)

1. Professional Email Draft

  • Purpose: Draft a polished email quickly
  • Prompt:

    You are a professional communications specialist. Draft an email to [recipient/role] regarding [topic]. Tone: [formal/friendly/urgent]. Include these key points: [point 1, point 2, point 3]. Keep under [X] words. Close with a clear call to action.

  • Output: Ready-to-send email
  • Tips: Specify the relationship (client, manager, vendor) for better tone calibration.

2. Meeting Summary

  • Purpose: Turn meeting notes into a structured summary
  • Prompt:

    Summarize the following meeting notes into a structured format with: (1) Key Decisions, (2) Action Items with owners and deadlines, (3) Open Questions, (4) Next Steps. Use bullet points. Keep it concise ‚Äî no more than one page. Meeting notes: [paste notes]

  • Output: Formatted meeting summary
  • Tips: Include attendee names in the notes so the AI can assign action items correctly.

3. Difficult Conversation Prep

  • Purpose: Prepare for a sensitive workplace conversation
  • Prompt:

    I need to have a conversation with [role] about [situation]. Help me prepare: (1) An opening statement that is direct but empathetic, (2) Three key points I need to communicate, (3) Anticipated pushback and how to respond, (4) A constructive closing that focuses on next steps. Tone: professional, calm, solution-oriented.

  • Output: Conversation script with objection handling
  • Tips: Provide context about the relationship and desired outcome.

4. Presentation Outline

  • Purpose: Structure a presentation quickly
  • Prompt:

    Create a presentation outline for a [duration]-minute talk on [topic] for [audience]. Include: a compelling opening hook, [X] main sections with key talking points for each, data points or examples to include, and a strong closing with call to action. Format as a numbered outline with sub-bullets.

  • Output: Presentation skeleton ready for slides
  • Tips: Mention if you want a storytelling arc, data-heavy approach, or persuasive structure.

5. Executive Summary

  • Purpose: Condense a long document for leadership
  • Prompt:

    Summarize the following document into an executive summary for [C-suite/board/VP-level audience]. Structure: (1) One-sentence overview, (2) Key findings (3‚Äì5 bullets), (3) Implications for our business, (4) Recommended actions, (5) Key risks. Maximum 300 words. Use plain language ‚Äî avoid jargon. Document: [paste or describe]

  • Output: Executive-ready summary
  • Tips: Specify what leadership cares about most (revenue, risk, timeline).

6. Stakeholder Update

  • Purpose: Write a project status update
  • Prompt:

    Write a stakeholder update email for [project name]. Include: Status (On Track / At Risk / Delayed), Key accomplishments this period, Upcoming milestones, Blockers or risks (with mitigation), and any decisions needed from stakeholders. Keep professional and concise. Data: [paste project status info]

  • Output: Formatted status update
  • Tips: Include RAG status (Red/Amber/Green) for visual clarity.

7. Policy Communication

  • Purpose: Announce a new policy to employees
  • Prompt:

    Draft an internal announcement for [policy name]. Include: What is changing and why, What employees need to do differently, Timeline and effective date, Where to get more information, and an FAQ section with 5 likely questions.   • Output: Internal communication with FAQ
  • Tips: Highlight the "what's in it for me" for employees.

8. Cross-Cultural Communication

  • Purpose: Adapt messaging for international audiences
  • Prompt:

    Adapt the following message for [target culture/region]. Consider: communication style norms (direct vs. indirect), formality expectations, cultural sensitivities, and appropriate greetings/closings. Maintain the core message while adjusting delivery. Original message: [paste message]

  • Output: Culturally adapted communication
  • Tips: Specify the specific country or region, not just "Asia" or "Europe."

  • Purpose: Evaluate risks for a project or initiative
  • Prompt:

15. Root Cause Analysis

  • Purpose: Investigate why something went wrong
  • Prompt:

    Perform a root cause analysis for the following issue: [describe the problem]. Use the 5 Whys technique. Then provide: (1) Root cause statement, (2) Contributing factors, (3) Immediate corrective actions, (4) Long-term preventive measures, (5) Recommended process changes.

  • Tips: Include timeline of events and data points for more accurate analysis.
